# ClassicModels_SalesDashboard
The Classic Models Sales Dashboard is an interactive Power BI dashboard built using the Classic Models sample dataset. It provides comprehensive insights into sales performance, customer behavior, product performance, and business profitability. The dashboard enables users to monitor key business metrics, identify trends, and make data-driven decisions through interactive visualizations.

Analyze overall sales performance and net profit. Compare sales across countries and product lines. Provide interactive filtering for better business insights.
